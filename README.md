# euclid_qso_public

Public science portal for the Euclid high-z quasar team (QSO Working Package).

🌐 **Website:** https://euclid-pu-qso.github.io/euclid_qso_public/

## About

This repository hosts the public-facing science portal for the **Euclid QSO Working Package** — the team within the Euclid Consortium responsible for the detection, selection, and scientific exploitation of quasars observed by the [Euclid space telescope](https://www.euclid-ec.org/).

The portal covers:
- Science goals of the Euclid QSO programme
- Public data products and catalogues
- Key publications and results
- Team information and contact details

## GitHub Pages

The site is built as a static HTML page served via [GitHub Pages](https://pages.github.com/). It is automatically published from `index.html` on the `main` branch.
